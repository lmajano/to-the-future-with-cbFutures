# cbFuture Stages

```
    -x-F-x-F-x-------------x-F-x------ DATA
cb()           \           /
    ------------E-F-E-F-E-F----------- ERROR
```

* Every interaction is a : `x` or `F`
* You can be zig-zagging into each track
* Work with closures or CFC instance methods
* Work with native CFML constructs
